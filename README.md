# CRM Management System

## Description
A simple **CRM (Customer Relationship Management)** application to manage leads, customers, and support tickets.  
This project demonstrates real-world CRM workflow using frontend technologies.

## Features
- Login & Dashboard
- Lead Management
- Support Ticket System
- Responsive UI

## Tech Stack
- React.js
- JavaScript
- HTML, CSS, Bootstrap
- Java
- Spring Boot
- REST APIs

## How to Run
1. npm install  
2. npm start  
3. Open http://localhost:3000
